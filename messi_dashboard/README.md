# ⚽ Messi Goals Dashboard — Stadium Night

Dashboard Streamlit untuk menganalisis gol Lionel Messi.

## Cara menjalankan

```bash
pip install -r requirements.txt
streamlit run app.py
```

Pastikan `messi_all_goals.xlsx` berada di folder yang sama dengan `app.py`.

## Fitur
- Header emas + 3 kartu metrik (Total Gol, Musim, Kompetisi)
- 6 filter multiselect di sidebar
- Tabel data interaktif
- 7 chart Plotly (line, stacked bar, grouped bar, 3 heatmap, horizontal stacked bar)
- Tema "Stadium Night — Messi Legacy" (hijau lapangan + aksen emas, biru Argentina, merah Barça)
